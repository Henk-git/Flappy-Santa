# Flappy Santa – Christmas Edition 🎅🎮

A festive Java-based arcade game inspired by the classic Flappy Bird. This project was developed as part of a programming competition and achieved **1st place** with a grade of **2.0**.

## 🚀 Overview
Flappy Santa challenges players to navigate Santa Claus through a series of obstacles (chimneys/pipes) to save Christmas. The project focuses on clean **Object-Oriented Programming (OOP)** and efficient game-loop logic.

## ✨ Key Features
* **Game Loop Engine:** Implemented from scratch using Java Swing/AWT.
* **Object-Oriented Design:** Clear separation of concerns (Player, Obstacles, Game Logic).
* **Collision Detection:** Precise hit-box calculation for game-over triggers.
* **Event Handling:** Responsive keyboard controls for flight mechanics.
* **Festive UI:** Custom Christmas-themed sprites and backgrounds.

## 🛠 Tech Stack
* **Language:** Java
* **IDE:** IntelliJ IDEA
* **Version Control:** Git

## 📈 Achievement
This project was awarded **1st Place** in a Java Programming Competition. 
* **Grade:** 2.0

## ⚙️ Setup & Installation
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/HenrikJustUR/Flappy-Santa.git](https://github.com/HenrikJustUR/Flappy-Santa.git)
    ```
2.  **Open in IntelliJ IDEA:**
    * File > Open > Select the project folder.
3.  **Run the application:**
    * Locate the `Main` class and press `Shift + F10`.
